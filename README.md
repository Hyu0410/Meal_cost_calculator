# Meal_cost_calculator
출근 정보 파일(.xsl)을 이용한 식대 계산 자동화 서비스

# Requirements
- 출근 정보 파일(.xls/.xlsx)
- 출력 이름 순서 파일(.xls/.xlsx)

# Notes
- 출근 정보 파일에는 `이름`, `출근`, `퇴근`이 명시되어 있어야 합니다. 이때 출퇴근 중 하나라도 값이 존재하면 출근으로 인정합니다.
- 출력 이름 순서 파일에는 `이름`, `비용`이 명시되어 있어야 합니다. 
- 식대를 계산한 파일을 다운로드한 뒤에는 입력한 파일을 모두 삭제합니다. 재출력하려면 다시 파일을 업로드하세요.
- `xlsx` 라이브러리를 사용합니다. 현재 **보안성** 주의가 필요한 라이브러리로, `xls`와 `xlsx` 확장자 모두 사용 가능하도록 구현하기 위해 위 라이브러리를 사용했지만 개인적으로 사용하는 게 아니라면 사용을 권장하지 않습니다.
